# Smart-Home
A smart home system in the early stages of development. 

# Current Stage
This project is in the early stages of making automatic blinds. It is currently in version 1.0 and is using an arduino, a 28BYJ-48 stepper motor that is controls the movements of the blinds using two switches. 

# Future Plans
  - Upgrade the arduino to a esp32 to allow the system to wirelessly communicate.
  - Upgrade the stepper motor to one that has a stronger turing force.
  - Upgrade from PLA to PTEG for the support of the system.
